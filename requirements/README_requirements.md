To install the packages:
    - Use pip on requirements.txt, you may need to change tensorflow package for gpu version<br/>
    - Install autokeras from source, version 1.0.16.post1, [link](https://github.com/keras-team/autokeras/tree/1.0.16.post1)<br/>
    - Make two small changes in autokeras, see diff [link](autokeras_diff.txt)

We ran the code on the following system:<br/>
    - CentOS 8<br/>
    - cuda 10.1<br/>
    - cudnn 7.6.5<br/>
